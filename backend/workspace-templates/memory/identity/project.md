## 3) Claim → Evidence 判据（必须遵守）

> 例：你经常要补齐“Co(IV) / ClO₂ 的硬证据链”。  
> 这里把“哪些证据才算支持/不支持”写清楚，避免凭印象。

### 3.1 Claim：体系中生成了 ClO₂（或可检测的 ClO₂）

**建议证据组合（至少 2 类）：**
- DPD 显色（注明波长/校准/干扰排查）
- UV-Vis 特征吸收（如有）
- 头空间/离子色谱（如有）
- 关键空白/对照：无催化剂、无 NaClO2、暗/光、pH 对照

**不充分证据（单独出现不允许下结论）：**
- 只有“气味/颜色变化”描述
- 只有单次显色无空白/无校准

### 3.2 Claim：存在高价 Co(IV) 或 Co=O 参与 OAT/PCET

**建议证据组合：**
- 原位/准原位谱学（XAS/XPS/EPR 等，按实验条件）
- 探针（如 PMSO）+ 产物定量 + 空白/对照
- 淬灭剂矩阵（说明副反应可能性与判读逻辑）
- 反应动力学与 pH/同位素效应等间接证据（须写清“间接”）

**关键提醒：**
- **探针/淬灭剂结果必须配套空白与干扰排查**（否则容易被审稿人打回）
- “推测 Co(IV)”必须标注证据等级（强/中/弱）
